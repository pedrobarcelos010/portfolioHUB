## 🚀 Projetos

### 🎸 Afinador Digital Inteligente para Guitarra
**Disciplina:** Introdução à Computação — UniCEUB (Abril/2026)  
**Parceiro:** Kalleb Abreu

Sistema embarcado baseado no **ESP32** que:
- Captura o áudio das cordas via microfone MAX4466
- Processa a frequência fundamental com **FFT**
- Exibe a nota detectada e o desvio de afinação no **display OLED SSD1306**
- Suporta armazenamento de **presets personalizados** via Wi-Fi integrado

**Stack:** `C++` `ESP32` `FFT` `OLED` `Tinkercad`
